DOM Manipulatie

Webpagina's dynamisch aanpassen

Wat is JavaScript?

JavaScript is een programmeertaal die in browsers werkt. Het voegt interactiviteit toe aan websites, zodat gebruikers met de pagina kunnen omgaan.

Terwijl HTML de structuur biedt en CSS de opmaak verzorgt, zorgt JavaScript voor gedrag en interactiviteit.

Voorbeelden van wat JavaScript kan doen:

  • Knoppen reageren op klikken
  • Inhoud van pagina veranderen zonder te herladen
  • Validatie van formuliergegevens
  • Animaties en effecten
  • Data opslaan in de browser
  • Communicatie met servers

Het script tag

JavaScript toevoegen aan HTML

JavaScript toevoegen aan HTML

Er zijn drie manieren om JavaScript toe te voegen aan een HTML-pagina:

1. Inline JavaScript (in een element)

Direct in een HTML-element met het onclick attribuut:

<button onclick="alert('Hallo!')">Klik mij!</button>

2. Intern JavaScript (in de pagina)

In een <script> tag in de <head> of <body>:

<script>
  console.log('Dit is JavaScript!');
</script>

3. Extern JavaScript (apart bestand)

In een apart .js bestand dat je linkt:

<script src="script.js"></script>

Beste praktijk: Gebruik extern JavaScript voor grotere projecten - het maakt je code netter en herbruikbaar!

Variabelen

Een variabele is een container voor gegevens. Je gebruikt variabelen om waarden op te slaan en later te gebruiken.

Variabelen declareren

let voornaam = "Lucas";
let leeftijd = 16;
let isStudent = true;
const stad = "Amsterdam";

let = variabele die je kunt veranderen
const = constante die niet verandert

Gegevenstypen

  • String - Tekst: "Hallo", 'test'
  • Number - Getallen: 42, 3.14, -5
  • Boolean - Waar of onwaar: true, false
  • Array - Lijst: [1, 2, 3] of ["a", "b", "c"]
  • Object - Verzameling van data: {naam: "Lucas", leeftijd: 16}

Herbruikbare code blokken

Functies

Een functie is een herbruikbaar blok code dat je kunt aanroepen wanneer je het nodig hebt.

Functie defini√´ren en aanroepen

function begroeting(naam) {
  console.log("Hallo, " + naam);
}

// Functie aanroepen
begroeting("Lucas");  // Output: Hallo, Lucas
begroeting("Joris");  // Output: Hallo, Joris

Functie met return waarde

function sum(a, b) {
  return a + b;
}

let resultaat = sum(5, 3);  // resultaat = 8
console.log(resultaat);     // Output: 8

Arrow functions (moderne syntax)

const multiply = (x, y) => {
  return x * y;
}

// Korte versie
const divide = (x, y) => x / y;

Events en Interactiviteit

Events zijn dingen die gebruikers doen, zoals klikken, typen, of hoveren. Je kunt erop reageren met JavaScript.

Veel gebruikte events

  • click - Gebruiker klikt op element
  • mouseover - Muis beweegt over element
  • change - Waarde van input verandert
  • submit - Formulier wordt ingediend
  • load - Pagina is geladen
  • keydown - Toets wordt ingedrukt

Event listener toevoegen

const button = document.getElementById('myButton');

button.addEventListener('click', function() {
  alert('Button werd geklikt!');
});

Event listener met arrow function

button.addEventListener('mouseover', () => {
  button.style.backgroundColor = 'red';
});

Console en Debugging

De console is een handig hulpmiddel voor debugging. Je kunt berichten afdrukken en fouten opsporen.

Console opening

Druk op F12 of Ctrl+Shift+I (Windows) / Cmd+Option+I (Mac) en klik op het Console tabblad.

Console methoden

console.log("Dit is een normale bericht");
console.warn("Dit is een waarschuwing");
console.error("Dit is een fout");

let persoon = {naam: "Lucas", leeftijd: 16};
console.table(persoon);  // Toont data in tabel vorm

Veelgemaakte fouten:

  • Typos in variabelen of functienamen
  • Elementen die niet bestaan in HTML
  • Foute gegevenstypen (getal in plaats van string)
  • Vergeten semicolons (hoewel JavaScript er meestal mee kan omgaan)

Tip: Gebruik console.log() veelvuldig om te controleren wat je code doet!
