Responsive Design

Websites voor alle apparaten

Wat is Responsive Design?

Responsive Design betekent dat een website zich aanpast aan verschillende schermgroottes. Een responsieve site werkt perfect op een mobiele telefoon, tablet en desktop!

In plaats van vaste breedtes, gebruiken we:

  • Flexibele layouts (Flexbox, Grid)
  • Procenten in plaats van vaste pixels
  • Media queries voor verschillende schermen
  • Flexibele afbeeldingen

Waarom belangrijk? Meer dan 50% van het web verkeer komt van mobiele apparaten!

Mobile-First Design

Begin met mobiel, voeg desktop features toe

Mobile-First Approach

Met Mobile-First design begin je met de mobiele versie en voeg je features toe voor grotere schermen.

Basis CSS (mobile)

.container {
  width: 100%;
  padding: 10px;
  font-size: 16px;
}

Toevoegen voor tablets en desktop

@media (min-width: 768px) {
  .container {
    width: 90%;
    margin: 0 auto;
    font-size: 18px;
  }
}

Voordeel: Website laadt sneller en is optimaal voor alle schermen!

Media Queries: Het hart van responsief design

Media queries laten je CSS regels wijzigen op basis van schermgrootte, ori√´ntatie en andere eigenschappen.

Basis media query structuur

@media (min-width: 768px) {
  /* CSS regels voor tablets en groter */
  .container {
    width: 750px;
  }
}

@media (min-width: 1024px) {
  /* CSS regels voor desktops */
  .container {
    width: 960px;
  }
}

Veel gebruikte breakpoints

  • 0-576px: Extra small (telefoons)
  • 576-768px: Small (grote telefoons)
  • 768-992px: Medium (tablets)
  • 992-1200px: Large (laptops)
  • 1200px+: Extra large (desktops)

/* Mobile first approach */
.container {
  width: 100%;
  padding: 10px;
}

.grid {
  display: grid;
  grid-template-columns: 1fr;  /* 1 kolom op mobiel */
}

/* Tablet */
@media (min-width: 768px) {
  .grid {
    grid-template-columns: 1fr 1fr;  /* 2 kolommen */
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .grid {
    grid-template-columns: 1fr 1fr 1fr;  /* 3 kolommen */
  }
}

Responsieve afbeeldingen

Afbeeldingen moeten ook responsief zijn. Er zijn verschillende manieren:

Max-width op afbeeldingen

img {
  max-width: 100%;
  height: auto;  /* Behoudt aspect ratio */
}

Srcset voor verschillende breedte afbeeldingen

<img src="small.jpg"
     srcset="medium.jpg 768w, large.jpg 1200w"
     alt="Responsieve afbeelding">

De browser laadt de beste afbeelding voor het apparaat!
