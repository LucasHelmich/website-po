Data Opslaan

LocalStorage & persistentie

Wat is het DOM?

Het DOM (Document Object Model) is een representatie van je HTML-pagina als een boomstructuur. JavaScript kan deze boom manipuleren.

Als je HTML er zo uitziet: 

<html>
  <body>
    <h1>Hallo</h1>
    <p>Dit is een paragraaf</p>
  </body>
</html>

Dan ziet het DOM er als een boom uit met parent- en child-elementen. JavaScript kan elk deel van deze boom veranderen!

Selecteren

Elementen vinden en selecteren

Elementen selecteren

Als je een element wilt veranderen, moet je het eerst selecteren. Dit doe je met verschillende methoden:

1. getElementById() - Selecteren via ID

let element = document.getElementById('mijnID');
// Selecteert element met id="mijnID"

2. querySelector() - Selecteren via CSS selector

let element = document.querySelector('.mijnClass');  // Klasse
let element = document.querySelector('#mijnID');     // ID
let element = document.querySelector('p');           // Tag
let element = document.querySelector('div > p');    // Combinatie

3. getElementsByClassName() - Selecteren via klasse

let elementen = document.getElementsByClassName('mijnClass');
// Geeft alle elementen met die klasse terug!

4. querySelectorAll() - Alle matching elementen

let allePs = document.querySelectorAll('p');
// Selecteert alle <p> tags

Voorkeur: Gebruik querySelector/querySelectorAll - die zijn moderner en flexibeler!

Inhoud veranderen

Je kunt de inhoud van elementen veranderen op verschillende manieren:

innerHTML - HTML toevoegen

let para = document.getElementById('myPara');
para.innerHTML = "<strong>Nieuwe tekst!</strong>";

textContent - Alleen tekst

let para = document.getElementById('myPara');
para.textContent = "Dit is veiligere tekst";

Verschil:

  • innerHTML - Kan HTML tags bevatten (voorzichtig met veiligheid!)
  • textContent - Alleen platte tekst, veiliger

Stijlen

Kleuren, lettertypen en opmaak

Attributen en Stijlen wijzigen

Attributen wijzigen met setAttribute()

let afbeelding = document.getElementById('myImage');
afbeelding.setAttribute('src', 'nieuw-plaatje.jpg');
afbeelding.setAttribute('alt', 'Een nieuw plaatje');

Attributen rechtstreeks wijzigen

let link = document.getElementById('myLink');
link.href = "https://google.com";
link.title = "Ga naar Google";

Stijlen wijzigen

let element = document.getElementById('myElement');
element.style.color = 'red';
element.style.backgroundColor = 'yellow';
element.style.padding = '10px';
element.style.fontSize = '20px';

Let op: CSS eigenschappen met koppeltekens (background-color) worden camelCase in JavaScript (backgroundColor)!

Klassen beheren

De beste manier om stijlen te wijzigen is door CSS klassen toe te voegen/te verwijderen:

classList.add() - Klasse toevoegen

let element = document.getElementById('myElement');
element.classList.add('active');

classList.remove() - Klasse verwijderen

element.classList.remove('active');

classList.toggle() - Omschakelen

// Voegt 'active' toe als het er niet is, verwijdert het als het er wel is
element.classList.toggle('active');

CSS bestand

.active {
  background-color: green;
  color: white;
  padding: 10px;
}

Voordeel: Verdeel logica (JavaScript) en stijlen (CSS) netjes!

Elementen

Nieuwe elementen maken en toevoegen

Nieuwe elementen maken

createElement() - Nieuw element aanmaken

let newPara = document.createElement('p');
newPara.textContent = "Dit is een nieuw element!";
newPara.className = "highlight";

appendChild() - Element toevoegen

let container = document.getElementById('myContainer');
container.appendChild(newPara);  // Voegt aan het einde toe

insertBefore() - Op specifieke plaats toevoegen

let firstChild = container.firstChild;
container.insertBefore(newPara, firstChild);  // Voegt bovenaan toe

Compleet voorbeeld:

function addItem(text) {
  let newLi = document.createElement('li');
  newLi.textContent = text;
  
  let list = document.getElementById('myList');
  list.appendChild(newLi);
}

Elementen verwijderen

removeChild() - Verwijder de kindelement

let parent = document.getElementById('parent');
let child = document.getElementById('child');
parent.removeChild(child);  // Verwijdert child uit parent

remove() - Verwijder jezelf

let element = document.getElementById('myElement');
element.remove();  // Verwijdert zichzelf direct

innerHTML = "" - Alle kinderen verwijderen

let container = document.getElementById('myContainer');
container.innerHTML = "";  // Verwijdert alles erin

Praktisch voorbeeld: Todo-lijstje

Hier een volledig voorbeeld van DOM manipulatie in actie:

<!-- HTML -->
<input type="text" id="todo-input" placeholder="Voer taak in">
<button onclick="addTodo()">Toevoegen</button>
<ul id="todo-list"></ul>
// JavaScript
function addTodo() {
  let input = document.getElementById('todo-input');
  let text = input.value;
  
  if (text === '') return;  // Niet leeg toevoegen
  
  // Maak nieuw element
  let li = document.createElement('li');
  li.textContent = text;
  
  // Voeg aan lijst toe
  let list = document.getElementById('todo-list');
  list.appendChild(li);
  
  // Maak input leeg
  input.value = '';
}

Dit doet: Voegt tekst toe aan een lijst, maakt input leeg, houdt alles netjes!
