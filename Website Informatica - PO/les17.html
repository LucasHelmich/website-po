Data Structures

Lists en Tuples

Wat zijn Lists?

Lists zijn veranderlijke (mutable) datastructuren die meerdere waarden kunnen bevatten. Ze kunnen verschillende datatypes bevatten en zijn een van de meest gebruikte datastructuren in Python.

Lists maken

# Lege lijst
lege_lijst = []

# Lijst met getallen
getallen = [1, 2, 3, 4, 5]

# Lijst met strings
namen = ["Lucas", "Joris", "Emma"]

# Lijst met gemengde datatypes
gemengd = [1, "hallo", 3.14, True]

# Lijst met list constructor
lijst = list("Python")  # ['P', 'y', 't', 'h', 'o', 'n']

print(lege_lijst)
print(getallen)
print(namen)
print(gemengd)
print(lijst)

Indexeren van lists

fruit = ["appel", "banaan", "kers", "druif", "mango"]

print(fruit[0])   # Eerste element: appel
print(fruit[2])   # Derde element: kers
print(fruit[-1])  # Laatste element: mango
print(fruit[-2])  # Een na laatste: druif

# Slicing
print(fruit[1:4])  # ['banaan', 'kers', 'druif']
print(fruit[:3])   # Eerste 3: ['appel', 'banaan', 'kers']
print(fruit[2:])   # Vanaf index 2: ['kers', 'druif', 'mango']
print(fruit[::2])  # Elke tweede: ['appel', 'kers', 'mango']

List Methodes

Bewerkingen op lists

List Methodes

Elementen toevoegen

fruit = ["appel", "banaan"]

# append() - voegt √©√©n element toe aan het einde
fruit.append("kers")
print(fruit)  # ['appel', 'banaan', 'kers']

# extend() - voegt meerdere elementen toe
fruit.extend(["druif", "mango"])
print(fruit)  # ['appel', 'banaan', 'kers', 'druif', 'mango']

# insert() - voegt element toe op specifieke positie
fruit.insert(1, "peer")
print(fruit)  # ['appel', 'peer', 'banaan', 'kers', 'druif', 'mango']

Elementen verwijderen

fruit = ['appel', 'peer', 'banaan', 'kers', 'druif', 'mango']

# remove() - verwijdert eerste voorkomen van waarde
fruit.remove("peer")
print(fruit)  # ['appel', 'banaan', 'kers', 'druif', 'mango']

# pop() - verwijdert en retourneert element op index (standaard laatste)
laatste = fruit.pop()
print(f"Verwijderd: {laatste}")  # Verwijderd: mango
print(fruit)  # ['appel', 'banaan', 'kers', 'druif']

eerste = fruit.pop(0)
print(f"Verwijderd: {eerste}")  # Verwijderd: appel

# del statement
del fruit[1]  # Verwijdert element op index 1
print(fruit)  # ['banaan', 'kers', 'druif']

Andere handige methodes

getallen = [3, 1, 4, 1, 5, 9, 2, 6]

# index() - vindt index van eerste voorkomen
print(getallen.index(1))  # 1

# count() - telt hoe vaak een waarde voorkomt
print(getallen.count(1))  # 2

# sort() - sorteert de lijst (verandert origineel)
getallen.sort()
print(getallen)  # [1, 1, 2, 3, 4, 5, 6, 9]

# reverse() - keert volgorde om
getallen.reverse()
print(getallen)  # [9, 6, 5, 4, 3, 2, 1, 1]

# sorted() - retourneert gesorteerde kopie (verandert origineel niet)
namen = ["Lucas", "Joris", "Emma", "Anna"]
gesorteerd = sorted(namen)
print(gesorteerd)  # ['Anna', 'Emma', 'Joris', 'Lucas']
print(namen)       # ['Lucas', 'Joris', 'Emma', 'Anna'] (onveranderd)

Wat zijn Tuples?

Tuples maken

# Lege tuple
lege_tuple = ()

# Tuple met √©√©n element (let op de komma!)
enkel_element = (42,)

# Tuple met meerdere elementen
coordinaten = (10, 20)
kleuren = ("rood", "groen", "blauw")
gemengd = (1, "hallo", 3.14, True)

# Tuple zonder haakjes (tuple packing)
driehoek = 3, 4, 5

print(lege_tuple)
print(enkel_element)
print(coordinaten)
print(kleuren)
print(gemengd)
print(driehoek)

Tuple unpacking

# Basis unpacking
x, y = (10, 20)
print(f"x = {x}, y = {y}")  # x = 10, y = 20

# Unpacking met meerdere variabelen
naam, leeftijd, stad = ("Lucas", 16, "Amsterdam")
print(f"{naam} is {leeftijd} jaar oud en woont in {stad}")

# Extended unpacking
eerste, *midden, laatste = [1, 2, 3, 4, 5]
print(f"Eerste: {eerste}")    # 1
print(f"Midden: {midden}")    # [2, 3, 4]
print(f"Laatste: {laatste}")   # 5

Immutable eigenschap

lijst = [1, 2, 3]
tuple_var = (1, 2, 3)

# Lijst kan worden gewijzigd
lijst[0] = 99
print(lijst)  # [99, 2, 3]

# Tuple kan NIET worden gewijzigd
# tuple_var[0] = 99  # TypeError!

# Maar je kunt wel een nieuwe tuple maken
nieuwe_tuple = (99,) + tuple_var[1:]
print(nieuwe_tuple)  # (99, 2, 3)

Lists vs Tuples

Verschillen overzicht

Aspect List Tuple
Syntax [1, 2, 3] (1, 2, 3)
Mutable Ja ‚úì Nee ‚úó
Methodes Veel (append, remove, etc.) Weinig (count, index)
Geheugen Meer geheugen Minder geheugen
Performance Langzamer Sneller

Wanneer lists gebruiken

  • Data die vaak verandert
  • Collecties die groeien/krimpen
  • Wanneer je elementen wilt toevoegen/verwijderen
  • Voor lijsten met taken, boodschappen, etc.

Wanneer tuples gebruiken

  • Data die niet mag veranderen (constanten)
  • Co√∂rdinaten, RGB kleuren, database records
  • Dictionary keys (moeten immutable zijn)
  • Function returns met meerdere waarden
  • String formatting

List Comprehensions

Basis list comprehension

# Traditionele manier
kwadraten = []
for i in range(1, 6):
    kwadraten.append(i ** 2)
print(kwadraten)  # [1, 4, 9, 16, 25]

# Met list comprehension
kwadraten = [i ** 2 for i in range(1, 6)]
print(kwadraten)  # [1, 4, 9, 16, 25]

Met voorwaarde

# Alleen even getallen
even_getallen = [x for x in range(1, 11) if x % 2 == 0]
print(even_getallen)  # [2, 4, 6, 8, 10]

# String filtering
woorden = ["appel", "banaan", "kers", "druif"]
lange_woorden = [woord for woord in woorden if len(woord) > 5]
print(lange_woorden)  # ['banaan', 'druif']

Geneste comprehensions

# Matrix maken
matrix = [[i * j for j in range(1, 4)] for i in range(1, 4)]
print(matrix)
# [[1, 2, 3],
#  [2, 4, 6],
#  [3, 6, 9]]

# Coordinaten
coordinaten = [(x, y) for x in range(3) for y in range(3)]
print(coordinaten)  # [(0,0), (0,1), (0,2), (1,0), (1,1), ...]

Praktisch voorbeeld

# Celsius naar Fahrenheit
celsius = [0, 10, 20, 30, 40]
fahrenheit = [(temp * 9/5) + 32 for temp in celsius]
print(fahrenheit)  # [32.0, 50.0, 68.0, 86.0, 104.0]

# Bestandsnamen filteren
bestanden = ["document.txt", "afbeelding.jpg", "script.py", "data.csv"]
python_bestanden = [f for f in bestanden if f.endswith('.py')]
print(python_bestanden)  # ['script.py']

Praktische toepassingen

# Studenten en cijfers
studenten = [
    ["Lucas", [8.5, 7.2, 9.1]],
    ["Joris", [6.8, 8.9, 7.5]],
    ["Emma", [9.2, 8.7, 9.8]]
]

# Gemiddelde per student
for naam, cijfers in studenten:
    gemiddelde = sum(cijfers) / len(cijfers)
    print(f"{naam}: {gemiddelde:.1f}")

# Speelveld (bijv. tic-tac-toe)
speelveld = [
    ["X", "O", "X"],
    ["O", "X", "O"],
    ["X", "O", "X"]
]

# Controleer winnaar
def check_win(board, player):
    # Controleer rijen
    for row in board:
        if all(cell == player for cell in row):
            return True
    # Controleer kolommen
    for col in range(3):
        if all(board[row][col] == player for row in range(3)):
            return True
    return False

print(check_win(speelveld, "X"))  # True
