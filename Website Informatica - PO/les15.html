Control Flow

Beslissingen en herhalingen

If-Else Statements

If-else statements worden gebruikt om beslissingen te nemen in je programma gebaseerd op voorwaarden.

Basis if statement

leeftijd = 16

if leeftijd >= 16:
    print("Je mag brommer rijden!")
    print("Gefeliciteerd!")

If-else statement

leeftijd = 14

if leeftijd >= 16:
    print("Je mag brommer rijden!")
else:
    print("Je bent nog te jong voor een brommer.")

If-elif-else statement

cijfer = 8.5

if cijfer >= 9.0:
    print("Uitstekend!")
elif cijfer >= 8.0:
    print("Goed!")
elif cijfer >= 7.0:
    print("Voldoende")
else:
    print("Onvoldoende")

Belangrijk: Python gebruikt indentatie (4 spaties) om code blokken aan te geven.

Comparison

Operators voor vergelijking

Comparison Operators

Vergelijkingsoperatoren

Operator Betekenis Voorbeeld
== Gelijk aan x == y
!= Niet gelijk aan x != y
< Kleiner dan x < y
<= Kleiner of gelijk x <= y
> Groter dan x > y
>= Groter of gelijk x >= y

Voorbeelden

leeftijd = 16
minimum_leeftijd = 16

if leeftijd >= minimum_leeftijd:
    print("Toegang verleend")
else:
    print("Toegang geweigerd")

naam = "Lucas"
if naam == "Lucas":
    print("Welkom terug, Lucas!")

cijfer = 7.5
if cijfer < 5.5:
    print("Herkansing nodig")
elif cijfer >= 8.0:
    print("Goed gedaan!")

Logical Operators

Logische operatoren

Operator Betekenis Voorbeeld
and Beide voorwaarden waar x > 5 and x < 10
or Minstens √©√©n waar x > 5 or x < 10
not Omgekeerde waarde not(x > 5)

Voorbeelden

leeftijd = 16
heeft_id = True

# and operator
if leeftijd >= 16 and heeft_id:
    print("Je mag naar binnen")

# or operator
dag = "zaterdag"
if dag == "zaterdag" or dag == "zondag":
    print("Het is weekend!")

# not operator
is_student = False
if not is_student:
    print("Je bent geen student")

# Complexe voorwaarden
cijfer = 8.5
aanwezigheid = 95

if cijfer >= 7.0 and aanwezigheid >= 80:
    print("Geslaagd!")
elif cijfer >= 6.0 and aanwezigheid >= 90:
    print("Geslaagd met extra voorwaarden")
else:
    print("Niet geslaagd")

For Loops

Herhalen over sequenties

For Loops

Basis for loop

for i in range(5):
    print(f"Nummer: {i}")

# Output:
# Nummer: 0
# Nummer: 1
# Nummer: 2
# Nummer: 3
# Nummer: 4

Range functie

# range(stop)
range(5)        # 0, 1, 2, 3, 4

# range(start, stop)
range(2, 8)     # 2, 3, 4, 5, 6, 7

# range(start, stop, step)
range(1, 10, 2) # 1, 3, 5, 7, 9

Loop over string

naam = "Python"
for letter in naam:
    print(letter)

# Output:
# P
# y
# t
# h
# o
# n

Loop met index

namen = ["Lucas", "Joris", "Emma"]
for i in range(len(namen)):
    print(f"Persoon {i+1}: {namen[i]}")

While Loops

Basis while loop

teller = 1
while teller <= 5:
    print(f"Telling: {teller}")
    teller += 1

# Output:
# Telling: 1
# Telling: 2
# Telling: 3
# Telling: 4
# Telling: 5

While met user input

antwoord = ""
while antwoord != "stop":
    antwoord = input("Typ 'stop' om te stoppen: ")
    print(f"Je typte: {antwoord}")

print("Programma gestopt!")

Infinite loop (voorzichtig!)

# Dit loopt voor altijd!
while True:
    print("Dit stopt nooit...")
    break  # Gebruik break om te stoppen

Break & Continue

Loop controle

Break en Continue

Break - stopt de hele loop

for i in range(10):
    if i == 5:
        print("Gevonden! Stoppen bij 5")
        break
    print(f"Nummer: {i}")

# Output:
# Nummer: 0
# Nummer: 1
# Nummer: 2
# Nummer: 3
# Nummer: 4
# Gevonden! Stoppen bij 5

Continue - slaat huidige iteratie over

for i in range(5):
    if i == 2:
        print("Sla 2 over")
        continue
    print(f"Nummer: {i}")

# Output:
# Nummer: 0
# Nummer: 1
# Sla 2 over
# Nummer: 3
# Nummer: 4

Praktisch voorbeeld

# Zoek eerste even getal
getallen = [1, 3, 5, 6, 7, 8, 9]
for getal in getallen:
    if getal % 2 == 0:
        print(f"Eerste even getal gevonden: {getal}")
        break

# Sla negatieve getallen over
getallen = [1, -2, 3, -4, 5]
for getal in getallen:
    if getal < 0:
        continue
    print(f"Positief getal: {getal}")

Nested Loops

Geneste for loops

for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")

# Output:
# i=0, j=0
# i=0, j=1
# i=1, j=0
# i=1, j=1
# i=2, j=0
# i=2, j=1

Praktisch voorbeeld: Vermenigvuldigingstafel

for i in range(1, 4):
    print(f"Vermenigvuldigingstafel van {i}:")
    for j in range(1, 11):
        print(f"{i} x {j} = {i*j}")
    print()  # Lege regel

Break in nested loops

for i in range(3):
    for j in range(3):
        if i == 1 and j == 1:
            print("Stoppen bij i=1, j=1")
            break
        print(f"i={i}, j={j}")
    else:
        continue
    break

# Output:
# i=0, j=0
# i=0, j=1
# i=0, j=2
# i=1, j=0
# Stoppen bij i=1, j=1
