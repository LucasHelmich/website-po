Python

Eenvoudig en krachtig programmeren

Wat is Python?

Python is een high-level programmeertaal die bekend staat om zijn eenvoud en leesbaarheid. Het werd ontwikkeld door Guido van Rossum in 1991.

Voordelen van Python

  • Eenvoudig - Makkelijk te leren en te lezen
  • Veelzijdig - Web development, data science, AI, automation
  • Grote community - Veel libraries en ondersteuning
  • Cross-platform - Werkt op Windows, Mac, Linux
  • Open source - Gratis te gebruiken

Toepassingen

  • Web development - Django, Flask
  • Data Science - Pandas, NumPy, Matplotlib
  • AI & Machine Learning - TensorFlow, PyTorch
  • Automation - Scripts voor dagelijkse taken
  • Game development - Pygame

Installatie

Python op je computer krijgen

Python Installeren

Om Python te gebruiken, moet je het eerst installeren op je computer.

Stappen voor installatie:

  1. Ga naar python.org
  2. Klik op "Downloads"
  3. Download de nieuwste versie voor je besturingssysteem
  4. Voer het installatieprogramma uit
  5. Belangrijk: Vink "Add Python to PATH" aan tijdens installatie
  6. Test de installatie in de terminal/commando prompt

Installatie controleren

Open een terminal/commando prompt en typ:

python --version

of

python3 --version

Je zou zoiets moeten zien: Python 3.x.x

IDE's en Editors

Integrated Development Environments (IDE's)

  • PyCharm - Professionele IDE van JetBrains
  • Visual Studio Code - Gratis editor met Python extensies
  • IDLE - Standaard Python editor (komt meegeleverd)
  • Jupyter Notebook - Voor data science en interactief programmeren

Onze aanbeveling: Visual Studio Code

  • Gratis
  • Veel extensies
  • Goede Python ondersteuning
  • Integrated terminal

Hello World

Je eerste Python programma

Eerste Python Programma

Hello World

Het traditionele eerste programma in elke programmeertaal:

print("Hello World!")

Uitvoeren:

  1. Open je editor
  2. Maak een nieuw bestand: hello.py
  3. Typ de bovenstaande code
  4. Sla het bestand op
  5. Open terminal/commando prompt
  6. Navigeer naar de map waar het bestand staat
  7. Typ: python hello.py

Wat doet deze code?

  • print() is een functie die tekst naar het scherm stuurt
  • Tekst moet tussen aanhalingstekens staan
  • Python-bestanden eindigen op .py

Python Syntax Basics

Belangrijke regels:

  • Python gebruikt indentatie (spaties) voor code blokken
  • Geen puntkomma's nodig aan het eind van regels
  • Statements eindigen automatisch aan het eind van de regel
  • Gebruik # voor commentaar

Voorbeeld:

# Dit is een commentaar
print("Hello")  # Dit ook

# Meerdere regels commentaar
# met hashtags

if True:
    print("Dit is ingesprongen")
    print("Dit ook")
print("Dit niet meer")

Python keywords

Woorden die je niet als variabelenaam kunt gebruiken:

and, as, assert, break, class, continue, def, del, elif, else, except, False, finally, for, from, global, if, import, in, is, lambda, None, nonlocal, not, or, pass, raise, return, True, try, while, with, yield

Oefening 1: Hello World

Oefening 2: Meerdere prints

  • Ik leer Python
  • Het is leuk!

Oefening 3: Commentaar

Voeg commentaar toe aan je code om uit te leggen wat elke regel doet.
