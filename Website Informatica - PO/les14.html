Variabelen

Data opslaan en gebruiken

Wat zijn Variabelen?

Variabelen zijn containers voor het opslaan van data waarden. In Python maak je een variabele door simpelweg een naam te geven en een waarde toe te kennen.

Variabele regels:

  • Beginnen met een letter of underscore (_)
  • Kunnen letters, cijfers en underscores bevatten
  • Hoofdlettergevoelig (age ‚â† Age)
  • Geen Python keywords gebruiken
  • Beschrijvende namen gebruiken

Voorbeelden:

# Goede variabelen namen
naam = "Lucas"
leeftijd = 16
is_student = True
_email = "lucas@email.com"

# Slechte variabelen namen
n = "Lucas"          # Niet beschrijvend
1naam = "Lucas"      # Begint met cijfer
naam-leeftijd = 16   # Streepje niet toegestaan
class = "Python"     # Keyword

Datatypes

Soorten data in Python

Python Datatypes

1. Strings (tekst)

naam = "Lucas"
bericht = 'Hallo wereld'
meer_regels = """Dit is een
string over
meerdere regels"""

2. Integers (gehele getallen)

leeftijd = 16
punten = 100
negatief = -5

3. Floats (komma getallen)

prijs = 19.99
pi = 3.14159
negatief_float = -2.5

4. Booleans (waar/onwaar)

is_student = True
heeft_auto = False
ingelogd = True

Type controleren

print(type(naam))        # <class 'str'>
print(type(leeftijd))    # <class 'int'>
print(type(prijs))       # <class 'float'>
print(type(is_student))  # <class 'bool'>

Type Conversie

String naar andere types

leeftijd_str = "16"
leeftijd_int = int(leeftijd_str)    # 16

prijs_str = "19.99"
prijs_float = float(prijs_str)      # 19.99

waar_str = "True"
waar_bool = bool(waar_str)          # True

Andere types naar string

leeftijd = 16
tekst = str(leeftijd)               # "16"

prijs = 19.99
tekst = str(prijs)                  # "19.99"

Let op met conversie!

# Dit werkt
int("123")     # 123
float("3.14")  # 3.14

# Dit geeft fouten
int("abc")     # ValueError
int("3.14")    # ValueError (gebruik float())

User Input

Gebruikersinput verwerken

Gebruikersinput

input() functie

Gebruik input() om tekst van de gebruiker te krijgen:

naam = input("Wat is je naam? ")
print(f"Hallo, {naam}!")

leeftijd = input("Hoe oud ben je? ")
leeftijd = int(leeftijd)  # Converteren naar integer
print(f"Volgend jaar ben je {leeftijd + 1} jaar oud.")

Voorbeeld programma:

print("Welkom bij de leeftijd calculator!")

naam = input("Wat is je naam? ")
geboorte_jaar = input("In welk jaar ben je geboren? ")

# Converteren naar integer
geboorte_jaar = int(geboorte_jaar)
huidig_jaar = 2024
leeftijd = huidig_jaar - geboorte_jaar

print(f"Hallo {naam}, je bent {leeftijd} jaar oud!")

Belangrijk: input() geeft altijd een string terug. Gebruik int() of float() voor getallen.

String Operaties

String concatenatie

voornaam = "Lucas"
achternaam = "Helmich"
volledige_naam = voornaam + " " + achternaam
print(volledige_naam)  # Lucas Helmich

String interpolatie (f-strings)

naam = "Lucas"
leeftijd = 16
print(f"Hallo, ik ben {naam} en ik ben {leeftijd} jaar oud.")

# Oude methodes
print("Hallo, ik ben " + naam + " en ik ben " + str(leeftijd) + " jaar oud.")
print("Hallo, ik ben {} en ik ben {} jaar oud.".format(naam, leeftijd))

String methodes

tekst = "Hallo Wereld"

print(tekst.upper())      # "HALLO WERELD"
print(tekst.lower())      # "hallo wereld"
print(tekst.title())      # "Hallo Wereld"
print(tekst.replace("Wereld", "Python"))  # "Hallo Python"
print(len(tekst))         # 12 (lengte)
print(tekst[0])           # "H" (eerste karakter)
print(tekst[-1])          # "d" (laatste karakter)
