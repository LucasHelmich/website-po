Wat is HTML?

HTML staat voor HyperText Markup Language. Het is de standaardtaal die wordt gebruikt om webpagina's te maken. Met HTML geef je structuur aan de inhoud van een website, zoals tekst, afbeeldingen en links.

HTML is geen programmeertaal, maar een opmaaktaal. Dat betekent dat je ermee beschrijft wat iets is, bijvoorbeeld een titel of een paragraaf, en niet hoe het eruitziet. Het uiterlijk wordt meestal geregeld met CSS.

Een browser, zoals Chrome of Firefox, leest de HTML-code en zet deze om in een webpagina die je kunt bekijken.

HTML en de browser

Een browser, zoals Chrome, Firefox of Edge, leest HTML-bestanden en zet ze om in een zichtbare webpagina. De browser leest HTML van boven naar beneden.

De browser begrijpt verschillende HTML-tags en weet daardoor welke onderdelen belangrijk zijn, zoals koppen, paragrafen en afbeeldingen.

Tags en elementen

HTML gebruikt tags tussen hoekige haakjes (< >). Samen vormen ze een element dat inhoud op een bepaalde manier markeert of structureert.

Voorbeeld van een paragraaf:

      <p>Dit is een paragraaf</p>
  • <p> is de openingstag
  • </p> is de sluitingstag
  • Alles ertussen is de inhoud

Sommige tags hebben geen sluiting, zoals <br> en <img>.

Basisstructuur van een HTML-pagina

Elke HTML-pagina begint met dezelfde vaste structuur. Deze structuur is nodig zodat de browser weet hoe de pagina is opgebouwd.

Betekenis van de onderdelen

  • <!DOCTYPE html> vertelt de browser dat dit een moderne HTML5-pagina is
  • <html> is het hoofdelement van de pagina
  • <head> bevat informatie over de pagina die niet zichtbaar is
  • <title> bepaalt de titel in het tabblad van de browser
  • <body> bevat alle inhoud die zichtbaar is op de pagina

Veelgemaakte fouten

  • Een sluitingstag vergeten
  • Tags verkeerd nesten, bijvoorbeeld een <p> in een <h1>
  • Het bestand opslaan als .txt in plaats van .html

          <!--Basis structuur van een HTML website-->
          <!DOCTYPE html>
          <html>
            <head>
              <title>Mijn eerste webpagina</title>
            </head>
            <body>
              <h1>Welkom</h1>
              <p>Dit is een eenvoudige webpagina.</p>
            </body>
          </html>
