JavaScript Basics

De basis van web interactiviteit

Wat is JavaScript?

JavaScript is een programmeertaal die interactiviteit en dynamisme toevoegt aan websites. Met JavaScript kun je reageren op gebruikeracties, gegevens verwerken, en de pagina aanpassen zonder opnieuw te laden.

Zonder JavaScript zijn websites statisch en passief. Met JavaScript worden ze interactief en responsief op gebruikerinvoer.

Elke moderne website gebruikt JavaScript - van het valideren van formulieren tot het laden van inhoud dynamisch.

Variabelen

Containers voor data

Variabelen en Gegevenstypen

Variabelen zijn containers om gegevens op te slaan. In JavaScript kun je verschillende soorten gegevens opslaan.

Gegevenstypen

  • String: Tekst ("Hallo")
  • Number: Getallen (42, 3.14)
  • Boolean: Waar of onwaar (true/false)
  • Array: Lijsten van gegevens
  • Object: Complexere gegevens

Voorbeeld: Variabelen declareren

// String
let voornaam = "Lucas";
let woonplaats = "Amsterdam";

// Number
let leeftijd = 16;
let gemiddelde = 7.5;

// Boolean
let isStudent = true;
let isVolwassen = false;

// Array
let klassen = ["1A", "1B", "2A"];
let nummers = [1, 2, 3, 4, 5];

// Object
let persoon = {
  naam: "Lucas",
  leeftijd: 16,
  stad: "Amsterdam"
};

Functies Schrijven

Functies zijn herbruikbare blokken code die een bepaalde taak uitvoeren. Ze helpen code georganiseerd en schoon te houden.

Waarom functies?

  • Code herbruiken - schrijf eens, gebruik meerdere keren

  • Makkelijker debuggen - fout in functie is snel gevonden
  • Betere leesbaarheid - functienamen vertellen wat ze doen

Events

Reageer op gebruikeracties

Events en Interactiviteit

JavaScript reageert op gebruikeracties door events: 

// Reageren op een klik
const button = document.getElementById('myButton');
button.addEventListener('click', function() {
  console.log('Button werd geklikt!');
  alert('Hallo!');
});

// Reageren op invoer
const input = document.getElementById('myInput');
input.addEventListener('change', function() {
  console.log('Waarde: ' + this.value);
});

// Reageren op het laden van de pagina
window.addEventListener('load', function() {
  console.log('Pagina is volledig geladen');
  initializeApp(); // Start je applicatie
});

// Diverse event types
// 'click', 'change', 'keypress', 'mouseover', 'submit', 'load'

Eindopdracht les 8

Opdracht

Maak een eenvoudig JavaScript programma dat met gebruikerinvoer werkt.

Vereisten

  • Definieer minstens 3 variabelen
  • Maak minstens 2 functies
  • Reageer op gebruikeracties (klikken, invoer)
  • Pas het DOM aan op basis van input
  • Voeg console.log() toe voor debugging

Bonus

  • Maak de interface visueel aantrekkelijk met CSS
  • Voeg validatie toe voor gebruikerinvoer
  • Test je code in de browser console
