Layout in CSS

Box Model, Display Types & Flexbox

Het CSS Box Model

In CSS is elk element een rechthoekige doos. Om een goede layout te maken, moet je begrijpen hoe deze doos is opgebouwd:

  • Content: De tekst of afbeelding zelf.
  • Padding: De binnenruimte tussen de content en de rand.
  • Border: De omlijning van het element.
  • Margin: De buitenruimte die afstand cre√´ert tot andere elementen.

Tip: Gebruik box-sizing: border-box; om te voorkomen dat padding je element breder maakt dan gepland!


/* Voorbeeld Display */
.blok {
  display: block; /* Neemt hele breedte */
}

.tekst-stukje {
  display: inline; /* Blijft op dezelfde regel */
}

Display Eigenschappen

Met de display eigenschap bepaal je hoe elementen zich gedragen ten opzichte van elkaar:

  • Block: Begint op een nieuwe regel en neemt de volle breedte in (bijv. <div>, <h1>).
  • Inline: Neemt alleen de breedte van de inhoud in en dwingt geen nieuwe regel af (bijv. <span>, <a>).
  • Inline-Block: Staat naast elkaar, maar laat je wel een breedte en hoogte instellen.

Flexbox: De 'Layout King'

Flexbox gebruik je om items binnen een container moeiteloos uit te lijnen. Je activeert het met display: flex; op de ouder-container.

Belangrijke eigenschappen:

  • justify-content: Horizontale uitlijning (center, space-between).
  • align-items: Verticale uitlijning (center).
  • flex-direction: Items in een rij (row) of kolom (column) zetten.

.container {
  display: flex;
  justify-content: space-around;
  align-items: center;
  height: 100px;
}

Eindopdracht les 5

Opdracht

Ontwerp een moderne "Feature Sectie" waarbij je de basis van layout en spacing toepast.

Vereisten

  • Maak een container die 3 informatiekaarten naast elkaar zet met Flexbox.
  • Gebruik padding om de tekst in de kaarten ademruimte te geven.
  • Zorg voor afstand tussen de kaarten met margin.
  • Geef de kaarten een border en een achtergrondkleur.

Bonus

  • Zet de tekst in de kaarten exact in het midden (zowel horizontaal als verticaal).
  • Voeg een box-shadow toe voor een diepte-effect.
