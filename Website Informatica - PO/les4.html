CSS Les 4: Basis CSS

Wat gaan we leren?

Wat gaan we leren in les 4?

  • Wat CSS is en waar het voor wordt gebruikt
  • De drie manieren om CSS toe te voegen aan HTML
  • Hoe je elementen kunt stylen met CSS
  • Kleuren en achtergronden aanpassen
  • Tekst opmaken met CSS
  • Margin en padding gebruiken

Aan het eind van les 4 kun je:

  • Uitleggen wat CSS betekent
  • CSS correct toepassen op HTML-elementen
  • Kleuren, fonts en spacing aanpassen
  • Een webpagina stylen met CSS

Wat is CSS?

CSS staat voor Cascading Style Sheets en is een opmaaktaal die wordt gebruikt om het uiterlijk van webpagina's te bepalen. Met CSS kun je de stijl en lay-out van HTML-elementen aanpassen, zoals kleuren, lettertypen, marges en meer.

Terwijl HTML de structuur en inhoud van een webpagina definieert, zorgt CSS ervoor dat die inhoud er aantrekkelijk uitziet en goed gepresenteerd wordt.

CSS maakt het mogelijk om de opmaak van meerdere pagina's centraal te beheren, waardoor het eenvoudiger wordt om een consistente stijl te behouden op een hele website.

CSS koppelen aan HTML

Er zijn drie manieren om CSS toe te voegen aan een HTML-pagina:

1. Inline CSS

Direct in een HTML-element met het style attribuut. 

<h1 style="color: blue;">Hallo</h1>

2. Interne CSS

In een <style> tag binnen de <head> sectie.

3. Externe CSS

In een apart .css bestand dat wordt gekoppeld met een <link> tag. Dit is de beste manier!

Kleuren en achtergronden

Met CSS kun je kleuren en achtergronden van HTML-elementen aanpassen met de eigenschappen color (tekst) en background-color (achtergrond).

Kleurwaarden

  • Kleur namen: color: red; color: blue;
  • Hex codes: color: #FF0000; (rood)
  • RGB: color: rgb(255, 0, 0); (rood)

Margin en padding

Margin is de ruimte buiten de rand van een element. Padding is de ruimte binnen de rand van een element. 


/* Kleuren en achtergronden */
h1 {
  color: white;
  background-color: blue;
}

/* Margin en padding */
p {
  margin: 20px;
  padding: 10px;
  background-color: lightgray;
}

/* Lettertype aanpassen */
body {
  font-family: Arial, sans-serif;
  font-size: 16px;
}

Eindopdracht les 4

Opdracht

Maak een eenvoudige webpagina over jouw favoriete hobby. Gebruik HTML voor de structuur en CSS voor de opmaak.

Vereisten

  • Een titel en een korte beschrijving van je hobby
  • Minstens drie verschillende HTML-elementen (koppen, paragrafen, lijsten)
  • Pas CSS toe om tekstkleuren en achtergronden aan te passen
  • Gebruik margin en padding voor ruimte

Bonus

  • Voeg een afbeelding toe en style deze met CSS
  • Experimenteer met verschillende lettertypen
